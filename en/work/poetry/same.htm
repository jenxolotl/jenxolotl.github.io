same

same people same time same day
same down glance same
same trains reflection same
light on same concrete same
leak same face same look
same guards same lock same
step tickets announcements
warnings same dead same
pavement same torso same
tracks same rust same leader
same statement same violence
sadness sleep misery
all same not same
unending changeless ageless
same
